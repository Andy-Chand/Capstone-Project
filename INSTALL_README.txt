Welcome to the Team Radagast Project, follow these step to get your program installed and running:

1. Click the Radagast tree icon shortcut.

2. After Visual Studio has loaded the project press F5 to run.

3. If you wish to upload a fresh database press 'YES' on the first modal, if the database is already loaded then press 'NO'.

4. Select the user by clicking on one of the names. An incorrect selection will cause error alerts to appear in the the Alberta Map page.

5. If you do make an incorrect selection, refresh the page or click on the home button to bring up the modals again.
